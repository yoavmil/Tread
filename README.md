# תִּדְרֹךְ — Tread

An interactive map for exploring and tracking visited places across Israel.

## What it does

- Sign in with Google
- Browse Israeli places on an interactive map
- Mark places as visited and track your progress
- Suggest edits or new places

## Getting started

See [SETUP.md](docs/SETUP.md) for full setup instructions.

```bash
npm run install:all   # install all dependencies
npm run seed          # populate the database
npm start             # run backend (3000) + frontend (4200)
```
